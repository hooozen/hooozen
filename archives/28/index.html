CSS3 制作常见几何图形

发表于 2017-09-16 | 更新于 2021-02-26 | 分类于 CSS

  通过 CSS3 提供一些新的属性和标准，可以实现一些常见的几何图形。避免使用贴图等其他手段，减少网络请求和资源大小，同时自适应和缩放效果会更好。

CSS 的边框

  因为 HTML 的元素一般是占据矩形空间，或者由矩形拼接的空间（如由于内容过长而换行的行内元素）。所以除了矩形之外的几何图形实现一般要借助 border 属性来完成。现在来了解一下 CSS 的 border 属性。

  border 相关的属性只有 border-width border-style border-color 和 border-radius 等，同时可以在 border 后添加 top bottom left 或 right 来指定单边边框。如，border-top-width: 10px，仅指定上边框的宽度为 10 px。

  border-style 可选的值有 dotted dashed solid double groove ridge inset outset none，可以通过下面的例子大概区分一下：

  border-width 和 border-color 属性很简单，指定宽度和颜色。颜色可以使用所有 CSS3 的颜色格式，可以通过指定 transparent 使边框透明，在利用边框绘制多边形时，经常会用到透明边框。

  border-radius 是 CSS3 的新属性，为边框指定圆角的半径。可以通过 broder-top-left-radius: 2px 的格式指定特定的角。在使用 CSS3 绘制圆形或椭圆形相关的图形时经常用到。

椭圆形
